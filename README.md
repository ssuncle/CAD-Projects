# My CAD Projects

This repo is where I store all of the CAD projects I've worked on. Most of these designs were created for robotics, engineering projects, school assignments, and personal practice. I enjoy designing parts and turning ideas into working models, and this repository is a collection of that work.

## What's in Here?

You'll find things like:

- Robotics parts and mechanisms
- Custom-designed components
- Assemblies and prototypes
- 3D printable models

## Why I Made This Repository

I created this repository to keep track of my CAD work and show how my design skills have improved over time. Each project helped me learn something new about engineering, problem-solving, and the design process.

Thanks for checking out my work!
